英語あれこれ

自分が理解した範囲のことしか書けませんし、間違いもあるかもしれませんが、なかなか辞書だけでは解決しないことなどもあるので、記憶の整理のために色々と書いてみようと思います。なお、私が主に読んできたのは情報系の文書と特許関連の文書ですので、わりと硬めの表現が中心です。文学的な表現や口語的な表現などは対象外です。我ながら殺伐としていますが、よく考えたら、日本語でだって「あら、このワンピース素敵。色違いがあれば見せていただける?」なんて店員と会話するのが無理なんだから (服屋ではなるべく店員が寄ってこないように祈り続ける派)、その手の「日常英会話」が私にこなせるわけがないのだ。というわけで、書き言葉のみを取り扱っています。

おかしな点などは @pi__yo__ko 宛に御教示くださいませ。

特定の形の表現を集めてみた
  • 「at + 名詞」
  • 「by + 名詞 (+ of)」
  • 「during + 名詞」
  • 「for + 名詞」
  • 「if 〜」
  • 「in + 名詞」 (一見、「in + 形容詞」の形に見えるものもある)
  • 「in + 名詞 + of」
  • 「in + 名詞 + with」
  • 「on (the) + 名詞 + of」
  • 「out of + 名詞」
  • 「over + 名詞」
  • 「to + 名詞」
  • 「with + 名詞」
  • 「with + 名詞 + to」と「without + 名詞 + to」
数に関すること
  • 数値的な大小の比較
  • 単数と複数の訳し分け
  • 数えるときに使う単位
  • 毎時、毎分
  • 不定の数が入っている表現 (n-th など)

  • 対になっている動詞

  • in と out で対になる言葉
その他
