英語あれこれ: 「if 〜」の形の言い回し

英語あれこれ: #if
if about to do「〜しようとするなら」
if and only if「もし〜ならば、かつ、その場合に限り」。数学などでよく使う。iff と略す。P if and only if Q は、P と Q が同値ということ。
if and when「もし〜するときには」「〜したらそのときには」
if any「もしあれば」「あるにしても (ごくわずか)」
if anything文脈によって色々に訳せる。「むしろそれどころか」「少しでもあるとして、一体何が〜?」「たとえあったとしても、ほとんどない」など。
if applicable「該当する場合」「妥当な場合」「当てはまる場合」
if desired「お望みなら」「希望するなら」「お好みで」「お好みなら」「必要なら」「必要に応じて」
if ever「仮にあるとしても」「皆無ではないにせよ」。
if in doubt「疑わしいなら」「不確かなら」「確信が持てないときは」
if necessary「必要なら」
if needed「必要なら」
if possible「可能なら」「できれば」
if required「必要なら」
little if any「あるとしてもごくわずか」
